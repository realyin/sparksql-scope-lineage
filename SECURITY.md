# Security Policy

Scope Lineage is a static SQL parser. It does not need database credentials and
does not execute SQL.

## Reporting Issues

If you find a security issue, please open a private advisory in the hosting
platform if available. If private advisories are not available, open an issue
with only minimal reproduction details and avoid posting sensitive SQL or
metadata.

## Data Safety

Before sharing examples or bug reports:

- remove private table names,
- remove personal data,
- remove company domains and email addresses,
- remove absolute local paths,
- replace production SQL with a minimized synthetic equivalent.
